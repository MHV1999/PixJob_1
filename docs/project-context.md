Project Name: PixJob

PixJob is a marketplace for creative services.

Users:
- Client
- Designer
- Moderator
- Admin
- Support 
- Finance
- Super_Admin

Core Features:
- Authentication
- User Profiles
- KYC Verification
- Projects
- Proposals
- Contracts
- Escrow
- Wallet
- Messaging
- Reviews

Tech Stack:

Frontend:
- Next.js
- TypeScript
- TailwindCSS
- shadcn/ui
- TanStack Query

Backend:
- NestJS
- Prisma
- PostgreSQL
- Redis

Architecture:
- Monorepo
- DDD
- SOLID
- Clean Architecture

Rules:
- Use UUID everywhere
- Use Prisma ORM
- Use PostgreSQL
- Use JWT Authentication
- Use Argon2 password hashing
- Support Persian and English
- Support RTL and LTR
